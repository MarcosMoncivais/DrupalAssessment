# _Half Priced Bookz_

##### _Bookstore Site in Drupal, 10/21/15_

#### By _**Marcos Moncivais**_

## Description

_This is a website I made using Drupal. This website allows users to share books they've read personally and allows for special "reviewer" members to submit reviews for books they've recently read!_

## Setup

* _Clone to your computer_
* _In phpmyadmin import database under the name assessment_database_
* _The database user name is: epicodus password: portland_
* _The drupal site username: epicodus password: portland_
